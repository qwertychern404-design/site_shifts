В таблице есть такие колонки:
1.     Worker
2.     Models
3.     Shifts
4.     Дни (пн-вс)

У таблицы есть 3 уровня отображения:

🔸Уровень Worker (свернутый) — Отображается только строка работника

🔹Колонка Worker
Имя работника.
Возможность развернуть список моделей.

🔹Колонка Models
Показывается количество моделей.
(Например: 3 models)

🔹Колонка Shifts
Общее количество смен за неделю по всем моделям работника.

🔹Колонки дней недели
Для каждого дня отображается суммарная информация по всем моделям и всем их аккаунтам:
- количество постов и ремувов
- количество смен (только просмотр, без возможности редактирования)

🔸Уровень Model (после раскрытия Worker) — раскрыто по дефолту при открытии данного раздела сайта
После раскрытия Worker под ним появляются строки всех моделей.

🔹Колонка Models
Имя модели.

🔹Колонка Shifts
Общее количество смен данной модели за неделю.

🔹Колонки дней недели
Отображается статистика уже только по аккаунтам данной модели:
 - количество постов и ремувов,
 - возможность установить количество смен — выпадающий список для выставления смен: 0, 0.5, 1, 1.5, 2

🔻Именно значения, выставленные на уровне модели, участвуют в подсчете итоговых смен.🔻

Каждую модель также можно раскрыть

🔸Уровень Account (после раскрытия Model)
После раскрытия модели отображаются отдельные аккаунты.

🔹Для каждого аккаунта показывается информация в столбиках по дням недели:
- количество постов и ремувов,
- время первого и последнего поста на аккаунте,
При наведении на ячейку отображается дополнительная информация:
Karma;
прирост кармы относительно предыдущего дня;
статус аккаунта (бан или активный)
Если аккаунт находится в бане:
строка или соответствующая ячейка подсвечивается красноватым цветом;
дополнительно отображается бейдж/флаг BANNED


🟨 Смены выставляются только на уровне модели. Для каждого дня доступен выбор одного значения (0, 0.5, 1, 1.5, 2)

Model. Количество смен за неделю рассчитывается как сумма значений за все дни. Например: Пн (1), Вт (0.5), Ср (1), Чт (0), Пт (1), Сб (1.5), Вс (0) = 5 смен. В колонке Shifts отображается эта сумма

Worker. Количество смен рассчитывается как сумма смен всех моделей. Например: Isabella (5), Missy (4.5), Ren (6) = 15.5 смен
В свернутом состоянии Worker в колонке Shifts отображает эту сумму. В колонках дней недели отображается сумма смен всех моделей за соответствующий день.

🟩Всегда закреплены три первые колонки:
•  Worker
•  Models
•  Shifts
При горизонтальной прокрутке они остаются на месте.
Все колонки с днями недели прокручиваются горизонтально, если они не помещаются
